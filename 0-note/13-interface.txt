Interface
- Interface adalah cara lain melakukan deklarasi data selain menggunakan Type
- Lantas kalo begitu, apa bedanya Interface dan Type?
- Interface bisa dikembangkan dengan mudah dibanding dengan Type
- Oleh karena itu, kebanyakan untuk tipe data yang kompleks, kebanyakan programmer TypeScript akan menggunakan 
  Interface dibandingkan menggunakan Type

Readonly Properties
- Properties dapat kita jadikan readonly, yg artinya tidak bisa diubah lagi
- Kita bisa menggunakan kata kunci readonly pada attribute tsb
